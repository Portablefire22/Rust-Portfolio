The hope is to host this website :3

### Running

Just your typical Rust application

```bash
cargo run
```

Alternatively, the following command may be used to automatically rebuild upon file update:

```bash
cargo watch -x run 
```

#### Command not found?

```bash
cargo install cargo-watch
```

### Release

```bash
cargo build --release
```
